Memory Models for Virtual Humans

Survey of memory models for virtual agents and humans, from psychological foundations to computational architectures.

A survey of memory models for virtual agents and humans, tracing the topic from its psychological foundations — episodic, semantic and working memory — to computational design and evaluation of memory systems for artificial agents.

The survey is published in (Pecora et al., 2025).
