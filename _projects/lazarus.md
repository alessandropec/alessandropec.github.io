Project Lazarus is an **LLM-driven cognitive architecture** for an immersive **virtual patient** used in **healthcare communication training**. Trainees interact by voice with an embodied patient in **VR role-play**; the system maintains an internal **patient-state model** that shapes the patient's behaviour during the interaction, and provides **reflective session analysis** afterwards.

A **patent application is in preparation** for this work.
